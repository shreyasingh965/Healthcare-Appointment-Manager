# AI-Powered Healthcare Appointment Manager

## Features

- User Authentication (JWT)
- Role-Based Authorization
- Doctor Management
- Appointment Management
- Doctor Leave Management
- AI Pre/Post Visit Summary
- Email Notifications
- Medication Reminder System

## Tech Stack

Backend
- Node.js
- Express.js
- PostgreSQL
- Prisma ORM
- JWT
- OpenAI API
- Nodemailer

Frontend
- React (In Progress)

## Installation

```bash
npm install
```

Backend

```bash
cd backend
npm install
npm run dev
```

Frontend

```bash
cd frontend
npm install
npm start
```

## Environment Variables

Create `.env`

```
DATABASE_URL=
JWT_SECRET=
OPENAI_API_KEY=
EMAIL_USER=
EMAIL_PASS=
```
